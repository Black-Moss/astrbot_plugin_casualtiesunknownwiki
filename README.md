# 星露谷维基查询插件

星露谷物语中文维基查询插件，支持查询游戏物品、任务、村民、攻略等信息。
数据来源使用声明：该数据来源`stardewvalleywiki中文维基`，侵权请联系我。

## 功能

- `/wiki <关键词>` - 查询维基页面内容
- `/wiki search <关键词>` - 搜索相关页面

## 示例

- `/wiki 丢失的斧子` - 查询任务攻略
- `/wiki search 酿酒` - 搜索相关页面
或者直接对bbot说 “帮我查询下星露谷物语中xx的信息“，代码里已经配置了tool，bot可以通过调用函数的方式来进行查询。

## 缓存

插件会自动缓存查询结果，缓存有效期 24 小时。

## 依赖

- aiohttp
